Gabriel's theorem and Birational Geometry
$X,Y$ algebraic varieties

Theorem(Gabriel) $X \simeq Y \iff \text{Coh}(X) \simeq \text{Coh}(Y)$

Theorem(C.-Pirisi) $X \sim_c Y \iff \mathcal C^c(X) \simeq \mathcal C^c(Y)$

$\mathcal C^c(X) := \text{Coh}(X)/\text{Coh}_{\leq \dim X -c -1}(X)$

Theorem $X \sim_\text{bir} Y \iff k(X) \simeq k(Y)$

A few comments on Gabriel's theorem

  • Original proof works for noetherian schemes
  • Rosenberg (+Gabber) extend to quasi-separated schemes$^*$ (see Brandenburg)
    $^*$for non-noetherian $X$, use $\text{QCoh}(X)$
    for noetherian $X$, $\text{QCoh}(X) = \text{IndCoh}(X)$
  • False for stacks: $\C \times \C \not\simeq B\Z/2$
    (true if you remember $\otimes$)     #Tannakian
  • What about algebraic spaces? Tricky, as proofs rely on ringed spaces.
  • Gabriel-Rosenberg: $\mathcal A$ $\mapsto$ $\text{Spec}(\mathcal A)$ [ringed space]
    $\text{Spec}(\text{Coh}(X)) = X$
  • C.-Groechenig: $\mathcal A$ $\mapsto$ $\text{Pt}_{\mathcal A}$ [moduli space]
  • Result is true for (quasi-compact and separated) algebraic spaces
What is $\text{Pt}_{\mathcal A}$?

Moduli space of points (or minimal objects)

$E \in \mathcal{A}$ is minimal if it has no sub-objects

  • $\text{Pt}_{\text{Coh}(X)} = X$? ...not quite
  • $\text{Pt}_{\text{Coh}(X)} = X \times B\mathbb G_\text{m}$
  • $\text{Pt}_{\text{Coh}(X,\alpha)} = \alpha$

Theorem $\text{Coh}(X,\alpha) \simeq \text{Coh}(Y,\beta) \iff \alpha \simeq \beta$

Conjectured by Caldararu, see previous work of Perego, Canonaco-Stellari, Antieau
